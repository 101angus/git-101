## 課程目的
- 能夠使用 git 而不手忙腳亂
- 能夠在本機端使用 CLI 介面操作 git
- 能夠使用 Github 的 pull-request 功能回饋更新後的原始碼
- 能夠解決衝突與建立/合併分支
- 能夠更新來自 upstream 的原始碼，並且發送 pull-request
- 能夠使用 git stash 指令暫存修改
