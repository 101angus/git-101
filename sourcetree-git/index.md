## 使用 SourceTree clone 專案

開啟 SourceTree，找到

- Windows 使用者找到
```
File -> New/Clone
2. 填入 Soruce URL
```
![](../img/sourcetree-git-1.png)


- Mac 使用者

```
1. New -> Clone from URL
2. 填入 Soruce URL
```
![](../img/sourcetree-git-1.png)
